## Project info
LIVE URL- https://spotless-schedule-wash.vercel.app/

## What technologies are used for this project?

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

With a clean and simple interface, you can: 

✅ See machine availability instantly from your room. 
✅ Book a guaranteed time slot in seconds. 
✅ Eliminate queues and arguments for good.
